# Cloudflare Tunnel na Kubernetes / Homelab portfolio

## How to:

1. Open cloudflared.yaml.
2. In `Secret` replace token placeholder on your real ZeroTrust CLoudflare tunnel token.
3. Apply the configuration to the kubectl:

```
kubectl apply -f .# portfoliopage
```
